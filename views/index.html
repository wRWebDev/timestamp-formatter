Timestamp Formatter

An API for converting timestamps between UTC and UNIX

For today's date and time, go to:
.../api/timestamp

To convert a UNIX or UTC timestamp, insert them after the timestamp query:
.../api/timestamp/YOUR_TIMESTAMP

All data returned in JSON format. e.g:
{"unix":1603187179778,"utc":"Tue, 20 Oct 2020 09:46:19 GMT"}

Errors handled by returning a JSON object:
{"error": "Invalid Date"}
